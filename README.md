[Awesome Advent of Code](https://github.com/Bogdanp/awesome-advent-of-code)

## Run & Test

Make sure you have installed [.Net 6.0](https://dotnet.microsoft.com/download/dotnet/6.0)

```powershell
# Run console program displaying all solutions
dotnet run

# Run tests
dotnet test
```
